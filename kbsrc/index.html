KBSRC Extension for Firefox

  • 在 KBS Web 网页中添加已读记录功能。
  • 支持 Firefox 内置的扩展更新检查功能。
  • 激活时在状态栏显示水木社区图标。
  • 登录时自动从服务器中读取全站所有公开版面已读记录。
  • 捕捉网页中已读记录相关的处理点，完全在客户端处理和显示公开版面已读记录。
  • 每十分钟将已读记录写回服务器。
  • 捕捉注销链接的点击，保证注销前将已读记录写回服务器。因为这个原因，点“离开本站”注销登录时，视网速浏览器会僵死一会儿。
  • 目前版本号 1.0.2。安装 下载
  • 有任何问题请在水木社区 Firefox 版讨论。
